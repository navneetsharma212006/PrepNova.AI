🚀 PrepNova.AI

PrepNova.AI is an AI-powered study assistant designed to help students learn smarter.
It provides AI chat support, document-based learning, quiz generation, and personalized study plans.

📌 Features
🤖 AI Study Assistant
Chat with AI for academic help
Ask questions from uploaded documents
Context-aware responses
📚 Document Learning
Upload subject notes
AI analyzes documents
Generate explanations and summaries
📝 Quiz Generator
Create MCQ quizzes from study materials
Instant scoring and feedback
📅 Study Planner
AI generates personalized study schedules
Helps manage time efficiently
